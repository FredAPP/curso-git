TUTORIAL GIT

Iniciando repositório

$ git init

Adicionando arquivo no índice

$ git add arquivo.txt

Commitando 

$ git commit -m 'mensagem'


Adionando repositório remoto

$ git remote add origin url

Enviando para repositorio remoto

$git push

Criando Branch

$ git branch develop


Mudando de branch

$ git checkout develop

Realizando merge

$ git merge master
